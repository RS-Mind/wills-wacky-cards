# Will's Wacky Cards

This is a mod containing a few cards I came up with while fiddling around with the game.

- Ammo Cache (Common)
- Slow Death (Rare)
- Vampirism (Rare)
- Shotgun (Uncommon)
- Basic Physics (Common)
- Alternate Universe Physics (Common)
- Minigun (Rare)
